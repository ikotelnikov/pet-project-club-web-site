Monte Guide

Интерактивный гид по Черногории с 1 250+ карточками услуг в 25+ категориях и охватом 15+ городов. Полная мультиязычность с AI-переводами, система отзывов, кабинет провайдера, премиум-подписки, букинг, чат с авто-переводом и ridesharing-сервис MonteCar.

active Next.js 16, React 19, TypeScript, Mantine, Tailwind CSS, Supabase (PostgreSQL, RLS, Realtime), Mapbox, OpenAI GPT-4o Святослав Твардовский

Подробности проекта

Работает как веб-приложение, PWA и Telegram Mini App.

Проект написан на Next.js 16 + React 19 + TypeScript с Mantine и Tailwind CSS, база данных — Supabase (PostgreSQL + RLS + Realtime), карты — Mapbox, AI-ассистент — OpenAI GPT-4o с web search.

Кодовая база включает 120+ API-эндпоинтов, 426+ E2E-тестов (Playwright), 45 accessibility-тестов и production-роадмап до Q1 2027.
